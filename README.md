## 🚀 Key Features

### 📊 1. Intelligent Dashboard
- **Real-time Stats:** Instant visibility into student strength, today's presence rate, and absence counts.
- **Attendance Trend:** Interactive 7-day trend visualization using high-performance HTML5 Canvas charts.
- **Top Attenders:** Recognition for students with the highest attendance percentage.
- **Class Analysis:** Class-wise distribution of attendance for comprehensive monitoring.

### 📝 2. Digital Attendance Marking
- **Smart Filters:** Instantly load students by Class and Section.
- **Interactive Cards:** Toggle attendance status (Present ✅ / Absent ❌) with beautiful visual feedback.
- **Bulk Operations:** Rapidly mark all students present or absent with a single click.
- **Safe Storage:** Real-time data persistence using `localStorage`.

### 📅 3. Attendance Register (Excel-Style)
- **Monthly Grid:** View the entire month's attendance in a clean, professional grid layout.
- **Excel Logic:** Automatic calculation of working days, presence, absence, and percentage per student.
- **Data Portability:** **Export to CSV** feature for backup and offline reporting.

### ✍️ 4. Academic Marks Management
- **Subject-Wise Entry:** Marks entry for Telugu, Hindi, English, Maths, Science, and Social.
- **Exam Types:** Manage Unit Tests, Mid-Term, and Final Exams.
- **Auto-Grading Engine:** Real-time percentage and grade calculation (A+, A, B+, B, C, F).

### 📜 5. Professional Report Cards
- **Instant Generation:** Beautifully formatted, printable student report cards with one click.
- **Comprehensive Data:** Includes institution headers, student details, subject marks, overall rank, and attendance summary.
- **Teacher Remarks:** Automated comments based on academic performance.
- **Print Optimized:** Clean, high-contrast layouts for physical printing.
